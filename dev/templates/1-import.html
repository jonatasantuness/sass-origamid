Utilizando @import no SASS

Esta página utiliza um arquivo CSS gerado através da compilação de vários arquivos .scss em um unico arquivo chamado 'main.scss'. Os arquivos que são importado podem ser nomeados com um underline '_' (também chamado de partials) na frente caso não seja necessário gerar um arquivo css para eles.

O SASS faz o import dos arquivos durante a compilação, assim o browser não precisa fazer vários imports e chama apenas uma requisição de arquivo css.
